# 💫 About Me:
Currently finalizing my Bachelor's thesis - an F-35 Lightning II flight simulator built in Unity - while contributing to EasyLessons, an educational platform developed with a small team. Always open to collaborating on web projects in modern JS/TS stacks (Next.js, React), as well as open-source contributions. Currently learning system design and leveling up my testing and CI/CD practices. Feel free to ask me about C#/Unity, Java or Python - I like exploring different corners of programming, from game dev to CS fundamentals.
